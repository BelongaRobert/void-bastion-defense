## Art Style Guidelines

### Tyranid
- **Colors**: Pink, red, purple (flesh tones)
- **Features**: Spikes, organic armor, multiple eyes
- **Vibe**: Alien horror, biological nightmare

### Ork
- **Colors**: Green skin, brown/orange clothing
- **Features**: Tusks, crude weapons, muscular build
- **Vibe**: Brutish, aggressive, ramshackle

### Chaos
- **Colors**: Purple, dark red, black
- **Features**: Spiked armor, glowing eyes, corrupted symbols
- **Vibe**: Dark, menacing, corrupted

## Recommended Sources:

1. **Kenney.nl** - Space shooter enemies as base
2. **OpenGameArt.org** - "alien", "monster", "orc" sprites
3. **Craftpix.net** - Free monster sprite packs

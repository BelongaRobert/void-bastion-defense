## Specifications

### Static Emoticons
- **Size:** 18x18 pixels
- **Format:** PNG transparent
- **Naming:** `{code_name}.png`

### Animated Emoticons
- **Size:** 18x18 pixels
- **Format:** APNG (Animated PNG)
- **Frames:** 8-16 frames
- **Duration:** 1-2 seconds per loop
- **Naming:** `{code_name}_animated.png`

## Style Guidelines

### Visual Consistency
- Maintain grimdark 40K aesthetic
- Consistent lighting style
- Same color grading across all shots
- No debug elements or UI glitches

### Composition Rules
- Action focused on golden ratio points
- Clear focal point in each shot
- Depth of field for cinematic feel
- No empty/dead space

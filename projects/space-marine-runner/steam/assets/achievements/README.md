## Art Direction Notes

### Style Guidelines
- **Theme:** Warhammer 40K inspired, grimdark, industrial
- **Color Palette:** 
  - Bronze/Gold for common achievements
  - Silver for uncommon
  - Gold/Platinum for rare
  - Imperial Aquila motif for milestones
- **Icons should feature:**
  - Space Marine helmets
  - Tower silhouettes
  - Enemy skulls/xenos heads
  - Imperial aquilas
  - Bolter shells
  - Defense structures

### Design Notes
- Achievements with progress bars should have visual stages
- Secret achievements should be masked/question mark in Steam UI
- Wave milestones should escalate in ornamentation (bronze → silver → gold → platinum → diamond)

# Steam Badges - Void Bastion Defense

Steam Community badges crafted from trading cards.

## Badge Progression

| Level | Name | Icon Description | XP Reward |
|-------|------|------------------|-----------|
| 1 | **Rookie** | Basic Imperial Aquila, bronze | 100 XP |
| 2 | **Defender** | Aquila with laurel wreath, bronze | 200 XP |
| 3 | **Veteran** | Aquila with crossed bolters, silver | 300 XP |
| 4 | **Champion** | Aquila with tower silhouette, gold | 400 XP |
| 5 | **Hero** | Aquila with full Bastion scene, platinum | 500 XP |

## Badge Specifications

### Icon Sizes
- **Small:** 54x54 pixels
- **Medium:** 64x64 pixels
- **Large:** 128x128 pixels

### File Naming
```
badge_level_{level}_{size}.png
```
Example: `badge_level_1_64.png`

## Badge Rewards

Each badge level unlocks:
- Background for profile (random from set)
- Emoticon (random from set)
- Coupon (chance for game/discount)
- XP toward Steam level

# DaShade Knowledge Base

Place your documents here to enhance DaShade's intelligence!

## Supported Formats:
- .txt (Plain text)
- .md (Markdown)

## How to Use:
1. Add files to this folder
2. Run DaShade (START.bat)
3. Documents auto-index on startup
4. Chat with /upload or just ask questions!

## Ideas for Content:
- Wikipedia articles (download as .txt)
- Technical manuals
- Personal notes
- Reference guides
- Books you've converted to text

⚔️ The Emperor protects!

- 👥 **Workers** — Staff list (no sensitive data)

**Implementation:**
- Clicking Payroll or Cosecha triggers password modal
- Password: **"Belonga"**
- Modal shows: "Acceso Restringido / Restricted Access"
- Unlock persists until logout
- Cancel returns user to previous tab
- Uses React state: `sensitiveUnlocked`, `showSensitivePrompt`, `sensitivePassword`

**Code Pattern:**
```tsx
const checkSensitiveAccess = (tab: Tab) => {
  if ((tab === 'payroll' || tab === 'cosecha') && !sensitiveUnlocked) {
    setShowSensitivePrompt(true)
    return false
  }
  return true
}
```

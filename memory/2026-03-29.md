## TwinSight Dashboard - Multi-Agent Architecture

**Location:** `~/.openclaw\workspace\projects\twinsight-dashboard`

**Purpose:** Real-time collaboration dashboard for AI agents (DaSage ↔ Uglyclaw)

**Architecture Established:**
```
┌─────────────┐ POST /api/events ┌─────────────┐ SSE /api/stream ┌─────────────┐
│  UglyClaw   │ ────────────────▶ │   Backend   │ ───────────────▶ │  Frontend   │
│  (Agent 1)  │                   │  (SQLite)   │                  │    (UI)     │
└─────────────┘                   └─────────────┘                  └─────────────┘
       ▲
       │ POST /api/events
┌─────────────┐
│   DaSage    │
│  (Agent 2)  │
└─────────────┘
```

**Components:**
- **Shared types:** `shared/types.ts` - Event contracts between agents
- **Backend:** Node.js + TypeScript + SQLite + SSE
- **Frontend:** Next.js 14 + TypeScript + Tailwind + shadcn/ui
- **Real-time:** Server-Sent Events over HTTP

**Event Schema:**
```typescript
interface EventSubmission {
  id: string;              // UUID (generated by agent)
  type: 'task_created' | 'task_updated' | 'file_modified' | 'command_run' | 'error';
  agent: 'uglyclaw' | 'dasage';
  timestamp: string;     // ISO 8601
  payload: TaskFrontmatter | FileEventPayload | CommandPayload | ErrorPayload;
  signature?: string;      // Optional: auth token
}
```

# 2026-03-23 — Farm Dashboard UI Polish Session

## Summary
Major UI improvements to the Cacao Farm Dashboard for mobile and dark mode support.

## Changes Made

### Header & Navigation
- **Darker brown header** - Changed from amber-400 → amber-800 for better visual hierarchy
- **Stacked buttons on mobile** - Dark mode toggle (👁️) now above language button (English/Español)
- **Centered title** - "Cacao Farm" now absolutely centered regardless of side buttons
- **Absolute positioning fix** - Prevents title shift when buttons change

### Dark Mode Feature
- **New toggle button** - Eye icon (open = light mode, closed/X = dark mode)
- **localStorage persistence** - Saves user preference across sessions
- **CSS dark mode support** added for:
  - All gray text levels (900→white, 800→light, 700→light gray, 500/600)
  - Card backgrounds (white → dark gray)
  - Page backgrounds (gray-50 → gray-900)
  - Border colors
  - Amber accent colors
  - Gradient backgrounds

### Mobile Responsiveness
- **Stat cards** - Reduced text size from 2xl to xl with `break-words` to prevent overflow
- **Profit numbers** - Added `break-all` for large currency values
- **Header layout** - Uses flex-col for buttons on narrow screens

### GitHub Pages Cleanup
- **Removed deploy workflow** - Deleted `.github/workflows/deploy.yml` to stop email notifications
- Dashboard deploys via Vercel only (already working)

### Previous Session Context
- Year comparison charts (2023/2024/2025) on Summary page
- Production vs Expenses card with 60/40 split estimate
- Payroll data integration from CSV
- Harvest detail views with weekly lot breakdowns

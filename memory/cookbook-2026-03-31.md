## 📚 Recipe Book

### Recipe 1: Enhanced Skills
**Location:** `~/.openclaw/workspace/skills/enhanced-skills/`

**What it does:**
- Defines skills using YAML frontmatter in `.md` files
- Auto-discovers from multiple sources
- Hot-reloads on file changes
- Estimates token usage

**Usage:**
```typescript
import { autoDiscoverSkills, getSkill } from './skills-system'

const skills = await autoDiscoverSkills()
const mySkill = getSkill('my-skill')
```

**Frontmatter format:**
```yaml
---
name: my-skill
description: What it does
whenToUse: When to use
features: [FEATURE_FLAG]
enabled: true
---
```

### Recipe 2: Memory Layers
**Location:** `~/.openclaw/workspace/skills/memory-layers/`

**What it does:**
- Organizes memory across 4 layers
- Auto-classifies entries
- Detects duplicates
- Suggests promotions

**Usage:**
```typescript
import { reviewMemory, promoteToLayer } from './memory-system'

const { promotions, duplicates } = await reviewMemory('working')
await promoteToLayer(entry, 'project')
```

**Layers:**
1. Working (`memory/YYYY-MM-DD.md`) — Daily logs
2. Project (`MEMORY.md`) — Shared conventions
3. Personal (`MEMORY.local.md`) — Private preferences
4. Skill — Tool preferences

---

### Recipe 3: Feature Flags
**Location:** `~/.openclaw/workspace/skills/feature-flags/`

**What it does:**
- Toggles capabilities without code changes
- Multiple sources (env, config, project)
- Conditional skill loading
- A/B testing support

**Usage:**
```typescript
import { isEnabled, enable } from './feature-flags'

if (isEnabled('MY_FEATURE')) {
  // Do something
}

enable('MY_FEATURE', true) // Persist to config
```

**Config:**
```json
{
  "features": {
    "MY_FEATURE": true,
    "BETA_FEATURE": false
  }
}
```

### Recipe 5: Voice Synthesis
**Location:** `~/.openclaw/skills/voice-synthesis/`

**What it does:**
- Generate voice from text using ElevenLabs
- Multiple voice options (Josh, Bella, Adam, Rachel)
- Adjustable settings (stability, clarity, speed)
- Play audio or save to file

**Usage:**
```typescript
import { synthesizeSpeech, playAudio } from './voice-synthesis'

const result = await synthesizeSpeech({
  text: "The Emperor protects!",
  voice: "Josh",
  stability: 0.5,
  clarity: 0.75
})

await playAudio(result.audioData)
```

**Config:**
```json
{
  "elevenlabs": {
    "apiKey": "sk-...",
    "defaultVoice": "Josh"
  }
}
```
